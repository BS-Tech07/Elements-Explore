About Elements Explorer

Elements Explorer ek modern aur interactive periodic table hai jo chemistry ko seekhne aur samajhne ka ek naya nazariya deta hai. Humne is project ko design kiya hai taaki students, teachers, aur science enthusiasts aasani se elements ke baare mein jaankari paa sakein.

Key Features

  • Interactive Filtering: Ek click mein metals, non-metals, aur gases ko alag karke dekhein.
  • Instant Search: Kisi bhi element ko uske symbol, atomic number, ya naam se turant search karein.
  • Detailed Data: Har element ke atomic mass, configuration, aur discovery ki complete history.
  • 3D Visuals: Atoms ko 3D mein explore karne ki suvidha.
  • Modern Interface: Dark mode aur responsive design.

Technical Stack

Frontend: HTML5, CSS3 , JavaScript
Data Handling: JSON Processing & Fetch API
3D Integration: <model-viewer> for Atomic Models
UI/UX: Responsive Grid & Fluid Layouts
